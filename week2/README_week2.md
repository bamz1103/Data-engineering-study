# Week 2: Customer Behaviour & Sales Analysis
### Project: Behavioral Data Modeling with PySpark

This project transitions from basic data profiling to **Behavioral Analytics**. I analyzed a retail dataset to identify patterns in customer purchasing habits, spending tiers, and product preferences.

## 🛠 Analytics Highlights: `Customer_Behaviour_Sales_Analysis`
The analysis focused on transforming raw transaction data into actionable business intelligence:

* **Customer Segmentation:** Aggregated purchase history to identify high-value customers based on total spend and frequency.
* **Product Affinity:** Analyzed which product categories drove the most engagement and revenue.
* **Trend Analysis:** Investigated sales performance over time to identify peak shopping behaviors.
* **Data Visualization:** Generated insights (exported as HTML) to communicate technical findings to non-technical stakeholders.

## 🚀 Key Skills Applied
* **Complex Joins:** Combining customer profiles with transaction logs to create a unified view.
* **Advanced Aggregations:** Using multi-level `groupBy` operations to slice data by demographics and time.
* **Data Storytelling:** Translating Spark DataFrame outputs into a readable analysis format.
